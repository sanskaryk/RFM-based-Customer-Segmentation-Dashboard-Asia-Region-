## 📊 RFM Based Customer Segmentation Dashboard (Asia Region)

### 🚀 Overview

This project showcases a **Power BI dashboard** built to analyze sales data and perform **customer segmentation using RFM analysis**.

It combines transactional sales data with customer behavior metrics to deliver meaningful business insights and support data-driven decision-making.

---

### 📁 Dataset Information

#### 1. Sales Data (`sales_asia.csv`)

* **country** → Country code
* **id** → Customer ID
* **week.year** → Time period
* **revenue** → Revenue generated
* **units** → Number of units sold

#### 2. RFM Data (`rfm_asia.csv`)

* **recency (R)** → Days since last purchase
* **frequency (F)** → Number of purchases
* **monetary (M)** → Total spending
* **rfm_score** → Combined score
* **segment** → Customer category (e.g., Loyal, Lost, Hibernating)

---

### 📊 Dashboard Features

* 📈 **Sales Performance Analysis**

  * Revenue trends over time
  * Units sold by region
  * Country-wise comparison

* 👥 **Customer Segmentation (RFM)**

  * Loyal Customers
  * Potential Loyalists
  * At-Risk Customers
  * Lost Customers
  * Hibernating Customers

* 🎯 **Key Insights**

  * Identify high-value customers
  * Detect churn risks
  * Track repeat purchase behavior

* 📍 **Regional Analysis**

  * Country-wise revenue distribution
  * Market performance comparison
